  1. loop kya hota hai? or ise ham q use krte hen.

Ye patterns ya structures programming mein commonly use hote hain aur particular tasks ko solve karne mein madad karte hain. or asan alfaz men jo kaam ap baar baar repeat krne wale hen use kam code or asani k saath repeat krne men loop apki madad krta hai. for example

    document.write(" hello world ");

    ab ye statement agar ap ko 10 baar print krwani hai to uske lye ap ise 10 times likenge but is kaam ko easy or fast banaya ja skta hai loop k zarie se wo is tarah k ap loop k zarie se ye aik code 10 times chala skte hen. is tarah se

    document.write(" hello world ");

     ye kaam kese krta hai age sab apko btaya jaega.s

    4. ★ for loop

  4. for loop ka syntax/formate kya hai?

    loop ka syntax/formate bhi if ki tarah hota hai. for example

    for(){
       code goes here
    }
    is men bhi usi tarah round Brackets men koi condition hogi or curly Brackets men wo code hoga jo hamne print krwana hoga. but is men kuch difference bhi hai or wo ye hai k for loop k round Brackets men sirf condition nhi lagai jati balke in round Brackets men 3 cheezen hoti hen.
    ★ initialization
    ★ condition
    ★ increament/decrement


    for example.
    for ( var i = 1; i <= 10; i++ )

  5. for loop k round Brackets men ane wali values ko thora explain kren?

    • initialization : iska mtlab hai for loop men sab se pehle aik variable initialize kya jata hai. for example
      for(var i = 0;)
    • condition : initialization k baad is variable k zarie ya kisi or tarah se condition lagai jati hai. for example
      for(var i = 0; i <= 10;);
      ab yhan condition lagai hai k agar i ki value 10 se kam hogi to age curly Brackets men dya gaya code chal jae.
    • increament/decrement : us k baad increament/decrement krwaya jata hai jis ka maqsad code execution ko repeat krwana hota hai. for example
      for(var i = 0; i <= 10; i++);
      ab code execution tab tak hogi jab tak i ki value 10 se kam hai or i ki value ko bharane k lye is increament ka use kaya gaya hai. loop aik baar chalega or phr increament hoga or ye circle tab tak chalega jab tak condition false na ho jae.

  6. loop men initialization k lye i name ka variable hi q banaya jata hai?

    iski waja ye hai k loop use hota hai iteration yani baar baar code dohrane k lye to is lye iteration ki munasibat se variable ka name bhi i se rakha hai. or ye koi zaruri nhi hai ap name kuch bhi rakh skte hen ye bas developers ki adat hai k wo i use krte hen.

  7. for loop ki working kese hoti hai?

    for loop ki working ko step by step address krte hen.
    for(var i = 1; i <= 5; i++)

    • sab se pehle initialization hoti hai yani for loop chalne per sab se pehle aik variable initialize hota hai or us men hamari btai huwi value chali jati hai. for example
      for(var i = 0;)
    • us k baad dusre number per loop men di huwi condition check hogi k wo true hai ya false. agar false huwi to loop whi ruk jaega or agar condition true huwi to loop 3rd step per chala jaega yani k code execute kr dega.
    • uske baad 4rth number per increament ya phr decrement hoga
    • uske baad phr condition check hogi agar true huwi to phr code execute ho jaega wrna ruk jaega.
    • uske baad dobara increament hoga or isi tarah tab tak ye circle chalega jab tak condition false nhi ho jati.

  8. kya ham for loop k zarie se array ko print krwa skte hen?

    g han ham for loop ki madad se array k elements ko one by one print krwa skte hen. for example

    var arr = ["jamal","kamran","sanaullah"]
    for(var i = 1; i <= 5; i++){
        document.write( i + ") " + arr[ i ] )
    }
    ab iska result aaega.
    0) jamal
    1) kamran
    2) sanaullah

    ye ese huwa k jab loop start huwa to i ki value thi 0 or hamne loop k block k andar loop k variable k is number ko array k index k tor per laga dya or ise print krwa dya to array k "0" index per jo element tha wo print huwa or phr dobara loop chala to i ki value increament ho k 1 ho gai to array k index 1 per mojood value print huwi isi tarah age bhi working hoti rhi
    or is men jo numbering shuru men print huwi wo bhi isi i ki values ko hi print krwaya huwa hai.

    9. Array Element Search Using Loop Flag Concept

    ★ loop flag

  9. flag kya hota hai? or ise q use krte hen.

    fleg aik variable hi hota hai normal sa but is men ham koi false value rakhte hen. for example
    var flag = false;
    take loop k zarie se array k kisi aik element ko search krne k lye. or ye kese krte hen age apko btaya jaega. or yaad rahe k variable ka name flag rakhna zaruri nhi hai.

  10. flag k zarie array elements ki searcing kese krte hen?

    loop zarie array elements ko ese search kr skte hen.

    • is k lye ap ko sab se pehle aik array banana hoga. for example
      var arr = ["sanaullah","jamal","kamran","raffay"]
    • uske baad aik flag(yani aik esa variable jis men koi false value hogi) banana hoga. for example
      var flag = false;
    • phr loop chala k us men condition laga kr batana hoga k hamari matlooba value ya element mile to hamen kya krna hai. for example

      for(var i = 0; i < arr.length; i++){
          if( arr[ i ] === "sanaullah"){
             flag = true;
          }
      }
      ab yhan per apni matlooba value ya element ko mojooda array men search kya ja rha hai. mil jane per block ka code run hoga or flag ki false value ko true kr dega agar value na mili to nhi krega.
    • us k baad ab loop k bahir aik if statement banai jaegi jis men ab ham condition falg ki basis per lagaenge k agar flag ki value flase ho to "else" chale or agar true ho to "if" ka code execute ho jae
      for example.
      if(flag == true){
             document.write("mil gai value")
      }else{
             document.write("nhi mili value") }

      to is tarah se array men se kisi bhi value ko search kya ja skta hai.

  11. flag k zarie array element ko search krne k code aik saath paish farmaen.

    ab yhan per flag Concept k zarie array element ko search krne ka code aik saath pais hai.

    • var arr = ["sanaullah","jamal","kamran","raffay"]
    • Var flag = False;
    • for(var i = 0; i < arr.length; i++){
          if( arr[ i ] === "sanaullah"){
             flag = true;
          }
      }
    • if(flag == true){
             document.write("mil gai value")
      }else{
             document.write("nhi mili value") }

    12. ★ break

  12. break kya hai? or q use hota hai.

    break aik keyword hai jo k sirf loops men hi use hota hai or ise loops ko break lagane k lye yani rokne k lye use kya jata hai. for example

    for(var i = 0; i < 10; i++){
        if( i == 5;){
           console.log( i )
          break
        }
    }

    ab yhan ye ho ga k loop chalega or jab i ki value " 5 " k barabar ho jaegi to loop whi ruk jaega. agar break na laga hota to if ki condition true hone k baad bhi loop apni condition false hone tak chalta but ab jese hi if ki condition true hogi loop whi per ruk jaega.

  13. break k zarie loop ko rokne ki kya zarurat?

    break se is lye loops ko roka jata hai take performance achi ho. q k jab ham break laga dete hen to loop uske baad age bilawaja nhi chalta jis se system ko kuch space mil jati or performance achi hoti hai.
