arrays

  1. array kya hote hen or inka syntax kya hai?

    array JavaScript ka aik data type hai
    or iska syntax/format ye hai
    var students = ["Sanaullah", "Raffay", "zohaib"]
    yhan per students jo hai wo aik array hai or ye "[ ]" square brackets array likhne ka structure/tareeqa/format hai ap isi tarah array men multiple values ko store kr skte hen.

  2. array ko q use krte hen?

    multiple values ko store krne k lye ya phr multiple value ka aik set banane k lye ise use krte hen or iski koi limit nhi hai ap jtne elements chahen store kr skte hen. for example
    apko students k name ki list chahie to ya to ap har student ka name alag alag variable men store kren jo k boht space bhi lega or lamba bhi hoga to is kaam k lye ham aik set banate hen jis men ham sari value ko jama kr dete hen jis se space bhi kam acquire hoti hai or tamam values ka aik set bhi ban jata hai. for example
    var students = ["sanaullah", "jamal", "kamran"]
    ab yhan per 3 students k name hamne aik hi jaga store kr k students k name ka aik set bana dya

  3. array ko print kese krte hen or agar array ki koi specific value chahie ho to wo kese print krte hen?

    array ko bhi ap usi tarah se print krenge jis tarah se ap variable ko print krte hen. for example
    var students = [ "Sanaullah", "jamal", "kamran" ];
    ab ap ise print is tarah se kr skte hen.
    document.write( students )
    iska result aaega kuch is tarah se.
    Sanaullah, jamal, kamran
    yani sari values hi print ho jaengi or agar ap kisi aik specific value ko print krana chahen to uske lye ap ko ye format use krna hoga.
    document.write( student[0] )
    yani array ka name likhne k baad jis tarah values assign krne k lye square brackets lagate hen wese brackets laga k jis number ki value apko print krwani hai wo number square brackets men dene se us index per mojood value print ho jaegi ab is example men sanaullah print hoga q k uska index 0 hai.

  4. ye number kis cheez ka apne lagaya hai array k name k baad?

    array tamam elements ko numberwise store krta hai or is ki numbring ko indexing kha jata hai yani har element ka aik number yani index hota hai or array men indexing 0 se start hoti hai yani k jo 1st element hai uska index/number 0 hoga or jo 2nd element hai uska index/number 1 hoga. for example
    var students = [ "sanaullah", "jamal", "kamran" ]
    ab is array men sanaullah ka index yani number 0 hai or jamal ka 1 or kamran ka 2 to agar ap is tarah se print krte hen.
    document.write( students[0] )
    to iska result hoga sanaullah
    q k ye 0 index per hai.

  5. agar hamne esa index dya jis per koi element na ho to kya print hoga?

    is situation men jese empty variable ko print krne per undefined milta hai isi tarah is index per value na hone ki waja se undefined aaega.

  6. agar ham kisi element ko update krna chahen or uski jag koi or value dena chahe to kya ye possible hai or esa kese krte hen?

    g ham array ki kisi bhi value ko update kr skte hen is tareeqa ye hai jis index per wo value hai jise apne update krna hai usi index per updated value assign krden. for example
    var students = ["sanaullah" , "jamal" , "kamran"];
    ab agar ap is array k 1st value ko update krna chahte hen to is tarah se kr skte hen.
    students[ 0 ] = "raffy";
    ab iska result hoga.
    raffy , jamal , kamran

  7. is men konsi type ki values ko store kya ja skta hai?

    string
    number
    object
    etc     sab ko ham array men store krwa skte hen. for example

    8. ★ adding & removing elements

  8. agar ham array men mazeed elements add krna chahen jese variable men baad men value update ya add ki jati hai to ye arrays men kese kr skte hen?

    iska boht asan sa tareeqa hai k jese ap kisi array element ko print krwate hen to array ka name likh k uske baad square brackets men wo index dete hen jis per mojood element apne print krwana hai to isi tarha jis index per apne element add krna hai us index ko array k name k saath laga kr aga value assign kr den. for example

    ab is array men 3rd index per aik or student ka name add ho gaya hai.

  9. agar hamen array ki length maloom na ho to uske akhir men kese element ko add krenge?

    iske lye hamare pas aik technich hai k ham pehle array ki length ko get krte hen or phr us length se milne wale number ko as a index number ham array k akhir men lagate hen jis tarah se ham array ki koi bhi value print krwate huwe index number likhte hen. for example

    var students = [ "sanaullah" , "jamal" , "kamran" , "zohaib" ]
    students[ 0 ];

    ab yhan per students k 1st index per jo bhi value hogi wo print ho jaegi or isi tarah se index bta kr ham koi bhi element add bhi kr skte hen .

    var students[4] = "wow";
    var students = [ "sanaullah" , "jamal" , "kamran" , "zohaib", "wow"]

    but masla yhi ata hai k hamesha hamen last index maloom nhi hota yani k array k andar ktne elements hen ye maloom nhi hota to ab is situation men ham last index per element ko lagane k lye length ka use krte hen. for example

    var students = [ "sanaullah" , "jamal" , "kamran" , "zohaib" ]
    students[ students.length ] = "wow"
    var students = [ "sanaullah" , "jamal" , "kamran" , "zohaib" , "wow"]

    ab yhan kya huwa thora sa wo samjh len. students ki indexing start hoti hai 0 se to is ka last index hoga 3 q k total elements 4 hen to last index 3 hoga jabke length bhi 4 hi hogi q k length to normal counting k hisab se hi number btati hai to students ka last index hoga 3 or next index hoga 4 or length hamen 4 hi return kr rhi hai q k total elements hi itne hen to
    students ka next index bhi 4 or length bhi 4
    to is tarah hamen array k last men element add krne k lye jo array k last men elements add krne k lye jo number chahie wo mil gaya jis k zarie ham akhir men elements add kr skte hen.

  10. agar array men 2 elements hun or ham index number 5 per aik or element add kren. for example

array methods

    ★ pop

  1. pop q use hota hai?

    pop kisi bhi array k last index ko remover krta hai or existing array ko update kr deta hai. for example
    var student = ["sanaullah" , "jamal", "kamran"]
    ab agar ise ham pop krte hen.
    students.pop
    to iska result hoga.
    sanaullah , jamal
    q k pop ne students k last wale index ko remove kr dya.

  2. pop kya koi value return krta hai or konsi value return krta hai? or pop ki returned value ko ham get kese kr skte hen.

    pop wo element return krta hai jise isne remove kya hota hai. for example
    var student = ["sanaullah" , "jamal", "kamran"]
    to ab is ki remove krda value ko get krne k lye jhan per hamne array ko pop kya hai us statement ko kisi variable men strore krna hoga take pop ki remove krda value us men store ho jae or phr ham use print kr sken. for example
    var returnedValue = students.pop
    console.log( returnedValue )
    to ab iska result hoga. kamran
    q k students wale array ki last value yhi hai.

    3. ★ length

  3. length q use hota hai?

    length array ka aik method hai jo k array k andar ktne elements hen ye btane k lye use hota hai yani array k andar total ktne elements hen ye maloom krne k lye is method ka use krte hen. for example .
    var students = [ "sanaullah", 23 , true ]
    ab yhan "students" men ktne elements hen ye maloom krna ho to ham is tarah se maloom kr skte hen.
    console.log( students.length )
    to ab result aaega 3 q k students k andar 3 elements hen
    note : ye baat bhi yaad rakhen k length ka indexing se koi talueq nhi yani k jab array ki values 5 hoti hen to last index to 4 hota hia q k array ki indexing 0 se start hoti hia to length bhi 0 se start ho esa nhi hai balke length 1 se hi start hoti hia jese normal counting hoti hai ye bas direct yhi btati hai k array k andar total elements ktne hen
