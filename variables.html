variables

  1. variables kya hota hai?

    variable aik tarah ka folder ya container type hota hai jis men ham apni marzi ki koi bhi value store kr skte hen?

  2. variable q banate hen?

    kisi bhi value ko store krne k lye

  3. variable kese banate hen?

    variable banane k lye JavaScript hamen aik key word deti hai jise ham jese hi likhte hen JavaScript ko pta chal jata hai k ham aik variable banane wale hen or wo keyword hai "var".

  4. kya ham variable ka koi name rakh skte hen?

    g han rakhh skte hen balke name rakhna zaruri hai jab ap "var" key word likhte hen to uske directly baad apko variable ka name likhna hota hai agar ap name nhi likhenge to variable nhi banega.

  5. variable men koi bhi value kese store krte hen?

    iska tareeka ye hai k ap jab variable ka name likhte hen to uske baad ap "equal to =" ka sign lagate hen or uske baad jo bhi value apne store krni ho to wo store ho jati hai.

  6. variable banane ka faida kya huwa?

    variable banane ka faida ye hai k ap bari se bari value ko is men store kr k use baar baar easily use kr skte hen or varibale ka use is lye bhi krte hen k koi bhi value jo baar baar use krni hia use easily use kya ja ske.

  7. variable ko print kese krte hen console men?

    console.log() men variable ka name directly likhne se wo console men print ho jata hai or document.write() men likhne se ye webpage men print ho jata hai but ismen khayal is baat ka krna hai k variable ka name inverted commas men na likha jae agar likha gaya to use variable nhi balke string samjha jaega to is lye iska name bina commas k likha jaega.

  8. variable k saath ham or konse operations kr skte hen?

    ham variable ko update bhi kr skte hen or iska tareeqa ye hai jhan variable ko define kya hai us se agli kisi bhi line men ap variable ka name likhe or uske baad equal to ka sign lagae phr jo bhi new wali value apne usmen update krni hia use likhde to variable update ho jaega. for example.
    var fullName = "Sanaullah";
    fullName = "Muhammad Sanaullah";
    ab yhan apne pehle variable men store kya sanaullah name or phr uske baad next line men use update kr dya to ab kya hoga k sanaullah ki jaga muhammad sanaullah ajaega.

  9. variable ka name kis case men likhte hen or cases ktne hote hen?

    koi bhi name likhne k lye 3 cases hote hen.
    ★ camalCase;
    ★ snack_case;
    ★ kibab-case;
    JavaScript men kisi cheez ka bhi (variable, function , object) ka name agar apne likhna ho to ap uper dye gye cases men se 2 cases ko use kr skte hen.
    ★ camalCase
    ★ snack_case
    iske ilawa 3rd case JavaScript men nhi chalta.

  10. usually JavaScript men konsa case use hota hai?

    JavaScript men camalCase; use hota hai kisi bhi (variable, function , object) ka name declare krne k lye jo snack_case hai wo jab databas per koi propery ya koi data bhejta hen to us men snack_case ka use krte hen.

  11. variable ka name kesa hona chahie?

    varibale ka name hamesha relivent hona chahie yani jo value usmen ap store krenge usi k hisab se uska name bhi hona chahie jese agar ap aik variable banate hen jis men ap user ka fullName rakhenge to is variable ka bhi fullName hona chahie take ise dekhte hi hamen pta chal jae k is men kya value hogi agar kisi ne irrelivent name bhi rakh dya to JavaScript to usko allow kregi but ye achi practice nhi hai.

  12. kya variable men numbers store kr skte hen or kya in numbers men mathametical operation kr skte hen?

    g han ham variable men numeric value bhi store kr skte hen or un men mathametical operation bhi kr skte hen for example.
    var num = 12;
    console.log(num + 3);
    iska result aaega 15.

  13. kya kisi number ko as a string store kr skte hen? or kya uske saath bhi mathametical operations kr skte hen.

    g han ham kr skte hen or iske saath mathametical operations bhi kr skte hen siwae "plus" krne k q JavaScript men "plus +" concatination k lye bhi use hota hai to jab ham ise 2 alag alag data type k saath use krte hen to JavaScript ise concatination k lye use krti hai. for example
    let num = "10";
    let result = num - 5;
    console.log(result);
     iska result hoga 5 q k chahe is baar 10 string men hai but uske saath minus wala mathametical operation huwa hai jo k ho skta hai. ab isi tarah agar ham ise plus kren for examplep
    let num = "10";
    let result = num + 5;
    console.log(result);
    to iska result hoga 105
     q k 10 to string men hai jo k jesa hai wesa hi rhega or 5 uske saath jur gaya to ye 105 ho gaya.

    ★ letalert

  15. variable ka name kin tareequ se likh skte hen?

    last men ya middle men number likh skte. for example
    ★ let name1st
    ★ let name1
