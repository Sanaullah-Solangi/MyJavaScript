strings

  1. string kya hota hai? or ise ham q use krte hen.

    string har programming language men aik data type hai. JavaScript or har programming language men ise text ko represnet krne k lye use krte hen JavaScript men number, boolean or variable k ilawa sab string men ata hai.

  2. string kese likhte hen?

    string likhne k lye ap text ko ("") inverted commas men likhte hen. for example.
    console.log("muhammad Sanaullah")
    yhan per name jo likha hai ye b hai

  3. agar string men number ko likha jae to kya wo bhi string ban jata hai

    g han agar ap number ko bhi string men likhte hen to wo bhi string ban jata hai. for example
    console.log("1234");

  4. agar string k letters ko index number k zarie get krna ho jese array k elements ko get krte hen to kya esa possible hai?

    g han string k letters ko bhi index number k zarie get kya ja skta hai. for example

    var str = "sanaullah";
    console.log(str[0])
    to result hoga "s";

    isi tarah ap index number k zarie string k andar se koi bhi letters get kr skte hen.

    ★ slice

  5. slice q use hota hai?

    slice basically existing string men se elements/letters copy kr k new string banane k lye use hota hai. yani agar ap kisi existing string men se kuch elements ko copy kr k aik new string men banana krna chahte hen to iske lye ap slice ko use kr skte hen. Jese kisi cake men se aik slice nikala jata hai same whi kam hota hai.

  6. slice return kya krta hai?

    slice ki return value ko pehle discuss krne ka maqsad ye hai k apko agle questions samajh men aa ske.
    slice wo elements return krta hai jo pehle wali existing string se copy kye hote hen. or copy krda values ko get krne k lye ap ko wo statement jis men apne pehle wali existing string men se kuch elements copy kye use kisi variable men store krna parta hai. for example

    q k 0 per "s" hai or 4 per "u" to "s" se lekar "u" se pehle tak copy ho jaega.

    or jab ham slice use krte hen to definately hamen return men elements chahie hoti hen isi lye iska use krte hen

  7. slice ko use krne ka tareeqa or syntax/formate kya hai?

    slice ka syntax/formate ye hai.
    students.splice(1,3)

    ab is men ham 2 values dete hen
    ★ 1st wo index number jhan se slice krna yani copy krna start krna hai.
    ★ 2nd wo index number jhan tak slice krna yani copy krna start krna hai. for example

    to ab yhan samjhne ki baat ye hai k jo ending point apne btaya hai ye us samet copy nhi krega balke us tak ya ese samjhen k us se pehle tak hi copy krega. jese k is example men "4" index yani k "u" se pehle tak hi copy kya hai.

    note : ye baat rakhen k minus men jo value ap denge slicing/coping us number se age hi hogi yani k esa nhi k minus men number denge to slicing/coping bhi ulti hogi wo usi direction men hogi jis men pehle ho rhi thi just indexing ka tareeqa hamne badla hai.
