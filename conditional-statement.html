  2. if statements q or kese use krte hen?

    if statement ka istemal condition ko check karne ya condition lagane aur uske mutabiq code ko execute karne ke liye hota hai. Agar aapko kisi shart ki tasdeeq karni hai, aur agar wo shart sach hai (true), to aap if statement ke andar ka code chalate hain.
    for example:

    • var time = 10 ;
    • if (time < 12) {
           console.log ("Good morning!"); // Agar 'time' 12 se chhota hai, to yeh print hoga
      };
    • Yahan, agar time 12 se chhota hai, to "Good morning!" print hoga. Agar time 12 ya usse zyada hota, to yeh code chalana band kar dega.
